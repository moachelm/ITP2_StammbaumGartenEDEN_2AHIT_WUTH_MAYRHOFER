# ITP2_StammbaumGartenEDEN_2AHIT_WUTH_MAYRHOFER

 Ausgangslage
Das Bestattungsunternehmen Garten Eden hat uns gebeten, eine Software zu entwickeln, mit der man einen Stammbaum darstellen kann.
Damit soll gezeigt werden, wie die einzelnen Familienmitglieder miteinander verwandt sind und welche Personen bereits von uns gegangen sind.
Vision
Die Software soll dem Unternehmen dabei helfen, den Überblick über die Familien zu behalten und diese Informationen auch den Angehörigen auf verständliche Weise zu zeigen.
